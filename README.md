# Tenxprotocols Tap

## How do I install these formulae?

`brew install tenxprotocols/tap/<formula>`

Or `brew tap tenxprotocols/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tenxprotocols/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
